Change button from radio selection

If you want to send the user to a different place based on a radio button you can do so by using 2 data attributes, "data-button-page" and "data-button-id" to say which page should be redirected to and which button should send the user there respectively.

For example, to redirect a user to "summary.html" when they select a checkbox with a label of "Summary" you can use the following: 

<label class="block-label" for="onward-1">
  <input id="onward-1" name="onwardPage" value="summary" type="radio" data-button-page="summary.html" data-button-id="#continuebutton" checked="checked">
  Summary
</label>

This looks for, when changed, a button with the ID "continuebutton" and changes it's href (where it goes when clicked) to "summary.html".

<a href="default.html" class="button" id="continuebutton">Continue</a>

If the radio button hadn't been changed, it'd still go to default.html instead.
