Toggle content with radio

If you'd like to toggle a content area based on which radio button or checkbox is selected, you can do so by giving the stuff you want to hide a class of "optional-section-(radio name)" and a attribute "data-toggle-value" of the radio or checkbox's value.

So, if you have a checkbox with a name of "travel-how" and a cheked value of "car" it would look like this:

<label class="block-label selectable" for="radio-1">
  <input class="selectable" id="radio-1" type="radio" name="travel-how" value="car" data-storage="Car">
  Car
</label>

If you now wanted to show a div when this radio button was checked, it would look something like this:

<div class="optional-section-travel-how" data-toggle-value="car">
  <p>You have selected car!</p>
</div>
